# Pi Day Pulse

Static one-screen Pi Day art object built with plain `HTML + CSS + JS`, `Canvas 2D`, and `Web Audio API`.

## Files

- `index.html`
- `style.css`
- `script.js`

## Local launch

Open `index.html` in a browser.

From WSL:

```bash
explorer.exe "$(wslpath -w /home/dmin/devel/private/_314159/index.html)"
```

Click or tap the scene once to start audio and pulse.

## Controls

- `Noise`: more grain, jitter, particles, and noise-audio
- `Pulse`: stronger and faster pulse
- `Slices`: number of pie sectors
- `Mode`: `Random` or `Quasi`
- `Play / Mute`: audio on or off after start
- `Piem`: opens the hidden text card

## Deploy

The project is ready for plain GitHub Pages publishing from the repository root without build steps. See `README_DEPLOY.md`.
