1. Create a GitHub repository.
2. Put `index.html`, `style.css`, `script.js`, `README.md`, `README_DEPLOY.md`, and `.nojekyll` in the repository root.
3. Commit and push to GitHub.
4. In GitHub, open `Settings -> Pages`.
5. Set source to your branch root, save, and wait for the site URL.
6. For updates, change the files and push again.
